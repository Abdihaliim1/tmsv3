---

## **Method 1: From Alerts Widget (Easiest)** ‚≠ê

### Step-by-Step:

1. **Go to Dashboard**
   - You'll see the "Alerts" widget on the left side

2. **Click "View" on any Missing POD alert**
   - The alert shows: "Load LD-2025-306 was delivered but POD is missing"
   - Click the blue **"View >"** link

3. **Load Edit Modal Opens**
   - You'll be taken to the "Loads" page
   - The load edit modal opens automatically
   - Scroll down to find the **"Documents"** section

4. **Upload POD**
   - In the "Documents" section, you'll see:
     - **"Proof of Delivery (POD) * Required"**
     - A drag-and-drop upload area
   - **Option A**: Drag your POD file into the box
   - **Option B**: Click the box to browse and select file
   - File types: PDF, JPG, PNG (up to 10MB)

5. **Done!**
   - Upload progress bar shows
   - Alert disappears automatically
   - Document is saved

---

## **Method 2: From Dispatch Board**

### Step-by-Step:

1. **Go to "Dispatch Board" page**
   - Click "Dispatch Board" in the sidebar

2. **Find the Load Card**
   - Look for the load with missing POD
   - It will have an alert icon (‚ö†Ô∏è) on the card

3. **Click "Upload POD" Button**
   - Purple button on the load card
   - Says "Upload POD"

4. **Select File**
   - File picker opens
   - Select your POD file

5. **Done!**
   - File uploads automatically
   - Alert clears

---

## **Method 3: From Loads Page**

### Step-by-Step:

1. **Go to "Loads" page**
   - Click "Loads" in the sidebar

2. **Find the Load**
   - Use search or filters to find the load
   - Look for loads with status "Delivered" or "Completed"

3. **Click Edit Button**
   - Click the pencil icon (‚úèÔ∏è) in the Actions column

4. **Scroll to Documents Section**
   - The modal opens
   - Scroll down past all the form fields
   - You'll see a section titled **"Documents"**

5. **Upload POD**
   - Under "Proof of Delivery (POD) * Required"
   - Drag file or click to browse
   - Select your POD file

6. **Done!**
   - File uploads
   - Modal can be closed
   - Alert will clear
